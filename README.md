# Blogs
*Веб-приложение для написания постов, с помощью markdown.*
Есть возможность добавления, редактирования и удаления постов в формате markdown. Организовано хранение исходных markdown в бд. Использован EntityFramwork. Заведены миграции. Посты могут иметь ссылки друг на друга в виде относительных ссылок. У поста есть название, категория, дата публикации. Предусмотрено отображение постов в ленте (сначала свежие). Для ленты посты отображены в режиме предпросмотра - название, дата, категории и начало (некоторая часть) самого поста.
